תגיות מתקדמות

בואו נלמד על תגיות מתקדמות יותר ב-HTML.

טפסים

טפסים משמשים לאיסוף נתונים מהמשתמש:

<form>
    <label for="name">שם:</label>
    <input type="text" id="name" name="name">
    <br>
    <label for="email">אימייל:</label>
    <input type="email" id="email" name="email">
    <br>
    <button type="submit">שלח</button>
</form>

<form> - תגית זו מגדירה טופס.

<label> - תגית זו מגדירה תווית לשדה טופס.

for - פרמטר זה מציין את מזהה השדה אליו מתייחסת התווית.

<input> - תגית זו מגדירה שדה קלט.

type - פרמטר זה מציין את סוג השדה (טקסט, אימייל, סיסמה וכו').

id - פרמטר זה מציין את מזהה השדה.

name - פרמטר זה מציין את שם השדה.

<br> - תגית זו מוסיפה מעבר שורה.

<button> - תגית זו מגדירה כפתור.

type - פרמטר זה מציין את סוג הכפתור (שלח, איפוס וכו').
